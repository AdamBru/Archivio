# Archivio

Aplikacja Archivio to system zarządzania archiwum w przystępny dla użytkownika i lekki dla systemu sposób.

## Główne funkcje aplikacji

- Widoki: Teczki, Spisy, Półki - umożliwiające wyświetlanie i edytowanie elementów 
- Wygodne formularze do dodawania i edycji elementów
- Funkcja eksportu i importu bazy danych
- Kompleksowy podręcznik użytkownika

## Technologia 

Aplikacja została stworzona przy pomocy [Tauri v2.0](https://tauri.app) (React + Rust).
