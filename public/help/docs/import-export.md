# Import, eksport i resetowanie

W [Ustawieniach](settings.html) aplikacji Archivio dostępne są opcje eksportowania i importowania pliku bazy danych oraz przywracania aplikacji do stanu początkowego.

## Eksport (kopia zapasowa)

Po klikniciu przycisku Eksportuj zostanie utworzona kopia bazy danych aplikacji w wybranej przez użytkownika lokalizacji.

## Import

Aby zaimportować bazę danych aplikacji Archivio z kopii należy wybrać przycisk Importuj. Po wybraniu pliku do importu dotychczasowe dane aplikacji zostaną przeniesione do pliku kopii zapasowej oraz podmienione przez te z importowanego pliku.

_[Jak odzyskać nadpisaną bazę dancyh?](problems.html#deleted-database)_

## Przywracanie stanu początkowego

Przycisk Resetuj umożliwia usunięcie wszystkich dancyh z aplikacji. Dla bezpieczeństwa dane są przenoszone do pliku kopii zapasowej.

_[Jak odzyskać usuniętą bazę dancyh?](problems.html#deleted-database)_
